Sharma sir's current reality
Before — phone chaos
Parents call during class to schedule visits. He misses calls, calls back during lunch, double-books slots, forgets appointments. Some parents give up and choose another center.
After — booking link
Parent clicks a link, picks a slot from available times, gets an automatic confirmation. Sharma sir sees all appointments in his calendar. No calls. No double bookings. No no-shows.
The student's role
Sharma sir is brilliant at teaching. He has no time to learn Calendly. The student sets it up, writes all the text using AI, and hands over one link. That's the service — and it directly stops him from losing potential students.
Live prompt to run in class
Paste this into ChatGPT or Claude
"Write the following for a Calendly booking page for Sharma Classes, a coaching center in Nagpur for Class 8–12 students: 1) A short warm description for the Parent Visit slot (20 minutes, in person). 2) A confirmation email the parent receives after booking. 3) A reminder message sent 1 hour before the visit. Keep everything under 80 words each, professional but friendly."
Teaching point
The student uses AI to write every piece of text on the booking page. Calendly handles the automation. The student's skill is knowing how to combine the two — and making the whole thing feel professional.
Setting up Calendly — step by step
1
Create a free Calendly account
Go to calendly.com and sign up free with Google. The free plan allows one event type — all Sharma sir needs.
calendly.com → Sign up free
2
Create the event type
Click "New Event Type" → One-on-one. Name it "Parent Visit — Sharma Classes". Set duration to 20 minutes. Paste the AI-written description.
New Event Type → One-on-one → 20 min
3
Set available hours
Set Sharma sir's visit hours — e.g. Mon, Wed, Fri 5–7 PM only. Parents can only book within these windows. Teaching hours are automatically blocked.
Availability → Set weekly hours
4
Paste the AI-written confirmation email
Add the address, what to bring, and what to expect. This goes automatically to every parent who books — zero effort from Sharma sir.
Notifications → Confirmation email → Edit
5
Share the booking link on WhatsApp
Copy the link and set it as a WhatsApp Business Quick Reply using the shortcut /visit. Any parent who asks to visit gets the link in one tap.
Share → Copy link → Add as /visit quick reply

If Calendly doesn't work — use this instead
Google Calendar appointment slots — backup option
1
Open Google Calendar
Go to calendar.google.com — 100% free with any Google account. No new tool to learn.
calendar.google.com
2
Create appointment slots
Click any time on the calendar → select "Appointment slots" instead of Event. Set duration to 20 minutes. Google splits the block into bookable slots automatically.
Click time → Appointment slots → 20 min
3
Get the booking page link
Google generates a booking page automatically. Parents can see available slots and book directly — no account needed on their end.
Appointment event → View booking page → Copy link
4
Share the link the same way
Add to WhatsApp Business as a /visit quick reply, Instagram bio, and Google Business listing. When parents book, both sides get a calendar invite automatically.
Calendly vs Google Calendar
Calendly looks more professional and is easier to customize — use it as the primary demo. Google Calendar is the backup if the owner already lives in Google and doesn't want another tool. Both are free. Both work perfectly.
